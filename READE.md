# 🏥 Smart Health System

A full-stack healthcare management system built with PHP and MySQL.

## Features
- 👤 User registration and login with sessions
- 📅 Appointment booking with double-booking prevention
- ⏱️ Live queue token system
- 💊 Medicine ordering with admin approval
- 📍 Hospital recommendation by health concern
- 🩺 Symptom checker with basic advice
- 🛡️ Admin dashboard — manage users, appointments, orders

## Tech Stack
- **Backend:** PHP 8
- **Database:** MySQL
- **Server:** Apache (XAMPP)
- **Frontend:** HTML, CSS, JavaScript (vanilla)

## Setup Instructions
1. Clone this repo into `C:\xampp\htdocs\smart-health\`
2. Import `smart_health.sql` into phpMyAdmin
3. Update `config/db.php` with your MySQL credentials
4. Start Apache and MySQL in XAMPP
5. Visit `http://localhost/smart-health/pages/register.php`

## Project Structure
```
smart-health/
├── config/        # Database connection
├── pages/         # User-facing pages
├── admin/         # Admin panel pages
└── assets/        # CSS and JS files
```
